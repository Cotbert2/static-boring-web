Preguntas Frecuentes

1. ¿Qué incluye la compra del software?

La compra del software incluye la licencia de uso, actualizaciones regulares (si están disponibles) y soporte técnico durante el periodo especificado en el plan adquirido. Algunos paquetes pueden incluir funcionalidades premium o módulos adicionales según el plan seleccionado.

2. ¿Cuáles son los requisitos mínimos del sistema para instalar el software?

Los requisitos mínimos pueden variar dependiendo del software. Asegúrate de que tu dispositivo cumpla con lo siguiente:

  1. Sistema operativo: (ejemplo) Windows 10, macOS 11 o posterior.
  2. Procesador: 2.0 GHz o superior.
  3. Memoria RAM: Al menos 4 GB.
  4. Espacio en disco: 2 GB libres.
  5. Conexión a internet: Requerida para la activación y actualizaciones.

3. ¿Es posible probar el software antes de comprarlo?

¡Sí! Ofrecemos una versión de prueba gratuita por 7 días. Durante este tiempo, puedes explorar todas las funciones del software antes de tomar una decisión.

4. ¿Qué sucede si no estoy satisfecho con el software después de la compra?

Si no estás satisfecho, ofrecemos una garantía de reembolso de 30 días desde la fecha de compra. Ponte en contacto con nuestro equipo de soporte para iniciar el proceso de reembolso.

5. ¿El software incluye actualizaciones gratuitas?

Dependiendo del plan que elijas, las actualizaciones menores son gratuitas. Las actualizaciones mayores (como nuevas versiones) podrían requerir un costo adicional, aunque ofrecemos descuentos especiales para clientes existentes.

6. ¿Cómo recibo el software después de comprarlo?

Después de realizar la compra, recibirás un correo electrónico con un enlace para descargar el software y una clave de activación para instalarlo en tu dispositivo.

7. ¿Cuántos dispositivos puedo usar con una licencia?

Cada licencia permite instalar el software en un dispositivo. Si necesitas usarlo en varios dispositivos, ofrecemos planes multiusuario con descuentos.

8. ¿Qué métodos de pago aceptan?

Aceptamos pagos a través de:

  1. Tarjetas de crédito/débito (Visa, Mastercard, American Express)
  2. PayPal
  3. Transferencias bancarias
  4. Otros métodos según tu ubicación
